# Fais moi une comparaison entre visual basic, VBnet et VBA dans in tableau. Je veux voir à quelle url aller chercher les informations, le prix, ce que l'on peut faire et ou aller chercher de l'information pour poursuivre mon apprentissage

Visual Basic (VB6), VB.NET et VBA sont des langages Microsoft liés au BASIC, mais avec des usages, architectures et coûts distincts ; consultez les tableaux ci-dessous pour une comparaison détaillée.[^1][^2][^3][^4][^5]

## Comparaison Principale

| Aspect | Visual Basic (VB6) | VB.NET | VBA (Visual Basic for Applications) |
| :-- | :-- | :-- | :-- |
| **Description** | Langage standalone pour apps desktop Windows, COM-based, fin de support officiel depuis 2008 [^2][^4][^5] | Langage .NET moderne, orienté objet, pour apps Windows/web/entreprise [^3][^6] | Macro-langage intégré aux apps Office (Excel, Word...), automation et scripts [^4][^5][^7] |
| **Ce qu'on peut faire** | Apps desktop avec drag-and-drop, bases de données (DAO/ADO), ActiveX, prototypes rapides [^2][^5] | Apps console/forms/web, LINQ, multithreading, services cloud, partage libs avec C\# [^3][^6] | Automatisation Office, macros Excel, rapports, traitement données sans apps externes [^8][^4][^7] |
| **Prix** | Gratuit (téléchargements legacy via Microsoft ou sites tiers comme Softonic), mais IDE obsolète [^9][^10] | Gratuit (Community edition Visual Studio) ; Express gratuit historiquement [^1][^11] | Gratuit, inclus dans Microsoft Office (abonnement Microsoft 365 ~7€/mois ou achat unique) [^8][^7] |
| **URL pour télécharger/info** | https://microsoft-visual-basic.fr.softonic.com ou https://www.clubic.com/telecharger-fiche210320-visual-basic-2010.html [^9][^10] | https://learn.microsoft.com/fr-fr/dotnet/visual-basic/ ou Visual Studio download [^1][^11] | Inclus dans Office ; https://learn.microsoft.com/fr-fr/office/vba/ [^8] |
